# Product Overview

QSB TLRS Calculator - Transport Lease Reimbursement Scheme calculator for Sabah government employees (Qhazanah Sabah Berhad).

## Purpose
Calculate monthly transport reimbursement based on:
- Grade-based daily rates (JG5-JGA: RM50-RM250/day)
- Automatic Sabah public holiday exclusion (fetched from sabah.gov.my)
- Sunday observed rule (holidays on Sunday observed next weekday)
- Personal leave deductions
- Weekend exclusion (Saturday/Sunday)

## Key Features
- GUI application with modern two-column layout
- Holiday caching for offline reliability
- Multiple export formats: CSV, Excel (calendar grid + details sheets)
- Print-ready A4 landscape Excel output
- Date validation and normalization
- Period navigation with Previous/Next month buttons

## Target Users
Sabah government employees calculating TLRS claims.
